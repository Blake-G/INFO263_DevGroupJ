for google maps to work:

index.php
add the following on line ~5 after the php and before the script
<div id="map"></div>

script/map.js
add the code (this is from the setup google api webpage):
function initMap() {
	var uluru = {lat: -25.363, lng: 131.044};
	var map = new google.maps.Map(document.getElementById('map'), {
	  zoom: 4,
	  center: uluru
	});
	var marker = new google.maps.Marker({
	  position: uluru,
	  map: map
	});
}

css/master.css
add the following code:
#map {
  height: 400px;
  width: 100%;
}
